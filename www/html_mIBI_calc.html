mIBI Calculator

Instructions

Download an .xlsx file from AWQMS with the site data for which you wish to calculate the mIBI values. Upload the document in this app using the "Upload..." button. The download button will remain deactivated until a file is uploaded.

Once the file has been successfully uploaded, the download button will be activated. Clicking the download button will prompt the mIBI calcultor to run. After a few seconds, an .xlsx file will be ready with the mIBI scores contained within. Feel free to name the file whatever you like and save it to the destination of your choosing.

Congratulations! You now have mIBI scores.
